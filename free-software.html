The four freedoms of free software

Free software is a matter of liberty, not price. Think of free as in free speech, not as in free beer.

Run

The user has the freedom to run the program as they wish, for any purpose (personal, educational, commercial, etc.).

Study

The user has the freedom to study the program and adapt it to their needs.

Distribute

The user has the freedom to distribute copies of the program, either gratis or in exchange for a price.

Improve

The user has the freedom to improve the program and distribute the improvements to others.

Free software licenses

There are many free software licenses, each with their own characteristics and benefits.

AGPL License

Allows the use, modification and distribution of the software, as long as the original source code is included. This license includes software that runs on servers.

GPL License

Allows the use, modification and distribution of the software, as long as the original source code is included.

LGPL License

Allows the use, modification and distribution of the software, as long as the original source code is included.

MIT License

Allows the use, modification and distribution of the software without restrictions, as long as the original copyright notice is included.
